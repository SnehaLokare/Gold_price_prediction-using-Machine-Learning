# Gold_price_prediction-using-Machine-Learning
Gold Price Prediction using Machine Learning A project to predict daily gold prices using historical data and financial indicators. Machine learning models analyze market trends, currency movements, and economic factors to provide accurate price forecasts for investors and analysts.

Dataset Overview

The dataset contains (2290,6) of historical financial data with the following columns:

Columns & Definitions

Date
The date of the record in MM/dd/yyyy format.

SPX
Represents the S&P 500 Index, one of the most widely used benchmarks of U.S. stock market performance.

GLD
Gold price indicator used as the target variable for prediction.

USO
Stands for the United States Oil Fund (USO), an exchange-traded fund that tracks crude oil prices.

SLV
Represents silver prices, used as a macro-economic indicator closely tied to gold trends.

EUR/USD
The Euro to US Dollar exchange rate, reflecting global currency market movements.

These financial indicators were selected because they have strong correlations with gold price fluctuations.
